Code for Toshima とは

街をもっと楽しく。その想いから Code for Toshima は生まれました。
自分たちの「暮らす街」「働く街」「遊ぶ街」。
そんな豊島区の「こうなればいいな。」を
テクノロジーを利用して皆で一歩一歩、進めていきます。
モットーは"正しく"より"楽しく"。
豊島区をより良くしたい。
そんな想いをもつ豊島区の行政、そこで働く皆さん、豊島区に「住む」「働く」「遊ぶ」私たちで「ともに考え、ともにつくる」を広げていきます。
